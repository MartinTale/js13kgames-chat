üöÄ Quick Start (Auto-Initialize)

Simply add the script and a container with data-chat-widget-auto attribute. The widget creates a floating chat button:

<!-- Add the chat widget script --> <script src="dist/widget/chat-widget.iife.js"></script> <!-- Add a container (can be body or any element) --> <div data-chat-widget-auto></div>

That's it! The widget will appear as a floating chat button in the bottom-right corner.

‚öôÔ∏è Manual Initialization (JavaScript)

Create a widget programmatically with custom options:

<!-- Add the chat widget script --> <script src="dist/widget/chat-widget.iife.js"></script> <script> // Initialize the chat widget const widget = window.createChatWidget({ container: document.body, // or any container element wsUrl: 'wss://relay.js13kgames.com/black-cat-chat' }); // Optional: Control the widget // widget.destroy(); // Remove the widget // widget.reconnect(); // Force reconnection </script>

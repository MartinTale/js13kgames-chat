üöÄ Quick Start (Auto-Initialize)

Add the script and a container with data-chat-widget-auto attribute. The widget fills the container:

<script src="dist/widget/chat-widget.iife.js"></script> <div style="width: 600px; height: 400px;"> <div data-chat-widget-auto></div> </div>

‚öôÔ∏è Manual Initialization (JavaScript)

Create a widget programmatically. Size the container as needed:

<script src="dist/widget/chat-widget.iife.js"></script> <div id="my-chat" style="width: 800px; height: 500px;"></div> <script> const widget = window.createChatWidget({ container: '#my-chat', wsUrl: 'wss://relay.js13kgames.com/black-cat-chat' }); </script>

üìè Different Sizes Example

The widget adapts to any container size. Here are some examples:

<!-- Small chat --> <div style="width: 400px; height: 300px;"> <div data-chat-widget-auto></div> </div> <!-- Large chat --> <div style="width: 100%; height: 600px;"> <div data-chat-widget-auto></div> </div>

üîß Integration Steps

  1. Download: Get chat-widget.iife.js from the dist/widget/ folder
  2. Include: Add the script to your HTML page
  3. Create Container: Add a sized container div where you want the chat
  4. Initialize: Use auto-init or manual JavaScript initialization
  5. Deploy: The widget fills its container and is ready to use!
